This webapp is about Online Advertisement Website, where users can register and post ads for sale.

To successfully run the application, do the following steps:
1 - Install database structure from "db_scripts/master.sql".
2 - Configure database in "model/db/dbconfig.ini".

The Application should run now smothly.

P.S. The workaround to send emails from localhost requires the xampp to be started as administrator.
